Core Components

The hardware stack behind the safety system

MQ-6 Gas Sensor

Detects LPG and other combustible gases with high sensitivity.

Microcontroller

Processes input values and controls the response sequence in real time.

Buzzer & LED

Provides audible and visual alerts once a leak is detected.

Relay / Valve

Cuts off the supply line or engages a shutdown protocol.

Wi-Fi / GSM Unit

Supports network communication for remote monitoring and messaging.

Dashboard Interface

Displays gas trends, warnings, and system status in an easy-to-read layout.
